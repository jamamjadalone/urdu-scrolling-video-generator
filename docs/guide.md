## Step-by-Step Tutorial

### Step 1: Choose Your Mode
- **Q&A Mode**: Best for educational, FAQ, or dialogue content
- **Story Mode**: Best for narratives, poems, or continuous text

### Step 2: Enter Your Content
- Type or paste your Urdu text
- Can use copy-paste from Word or other text editors
- Text formatting is preserved

### Step 3: Customize Colors
- Click color picker next to each text type
- Yellow (#ffff00) works well for questions
- Green (#00ff00) works well for answers
- White (#ffffff) works for dark backgrounds

### Step 4: Select Font
- **Jameel Noori**: Traditional, elegant
- **Nafees Naskh**: Clear, readable
- **Alvi Nastaliq**: Beautiful, decorative
- **Mehr Nastaliq**: Modern, sleek

### Step 5: Adjust Size & Spacing
- **Font Size**: 48px recommended (20-80 available)
- **Line Height**: 1.8 default (1-3 range)
- **Letter Spacing**: 0px default (0-10 range)
- **Text Margin**: 10px default (0-50 range)

### Step 6: Set Background
- Solid color: Black (#000000) is YouTube standard
- Image URL: Paste link to image (JPG, PNG, WebP)
- Leave blank for default color

### Step 7: Animation Speed
- Default: 30 (balanced)
- Slow: 15-20 (leisurely reading)
- Fast: 35-45 (quick paced)
- Recommended: 25-30 for most content

### Step 8: Optional Effects
- **Border Animation**: Glow effect around preview (enable/disable)
- **Resize Preview**: Drag corner to custom size

### Step 9: Setup & Record
1. Click "Setup Preview" button
2. Resize preview window if needed
3. Open your screen recorder (OBS, Camtasia, etc.)
4. Click "PLAY" when ready to record
5. Let animation complete
6. Stop recording

### Step 10: Edit & Upload
1. Edit in video editor (add music, effects, intro/outro)
2. Export as MP4
3. Upload to YouTube with title and description
4. Add thumbnail and tags
5. Enable monetization (if eligible)

## Advanced Customization

### Creating Professional Layouts

**Education Content:**
- Font Size: 52px
- Line Height: 2.0
- Q Color: Yellow (#ffff00)
- A Color: Green (#00ff00)
- Background: Dark blue
- Speed: 28

**Religious Content:**
- Font Size: 48px
- Line Height: 2.0
- Text Color: Gold (#FFD700)
- Background: Dark green or black image
- Speed: 25

**Entertainment Content:**
- Font Size: 56px
- Line Height: 1.6
- Multi-color (change for each line)
- Background: Colorful image
- Speed: 35

## Troubleshooting Guide

### Text Not Showing
- Solution: Change text color (too light for background)
- Solution: Increase font size
- Solution: Check Urdu text is valid

### Font Looks Strange
- Solution: Try different font from dropdown
- Solution: Increase font size
- Solution: Adjust letter spacing

### Preview Won't Resize
- Solution: Move cursor to bottom-right corner
- Solution: Drag corner handle to resize
- Solution: Check browser supports resize

### Scrolling Stutters
- Solution: Close other browser tabs
- Solution: Reduce animation speed
- Solution: Clear browser cache

### Recording Quality Poor
- Solution: Use higher resolution (720p+)
- Solution: Resize preview window larger
- Solution: Close background applications
